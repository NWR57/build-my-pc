About Us

Build My PC is an advanced discord bot made using the Discord.JS v13 API. This bot provides :

  • Gaming and Work PC builds from any range you want
  • Lastest tech news and updates
  • And many more fun commands

Build My PC bot is a bot made and powered by tech enthusiasts. This bot was made at first to give builds to our discord server members but later was made public so tech hungry people all around the world can have a taste of it. This bot is made by members of PC Builder Bangladesh. Know more about the devs here. You can also join our discord server if you want <3
